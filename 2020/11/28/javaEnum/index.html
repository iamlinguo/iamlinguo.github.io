我们再对枚举类做一些优化

public enum clothesType{

    JACKET("上装"),BOTTOMS("下装"),SHOES("鞋子");  //枚举值

    private final desc; // 枚举描述 对应括号中的内容  

    private clothesType(String desc){   //枚举的构造函数，我们使用枚举的时候，这个参数的值就是括号中的内容
        this.desc = desc;
    }

    // 通过枚举值即名称或描述来转换成枚举
    public static clothesType parse(String nameOrDesc){
        for(clothesType item : clothesType.values()){  //遍历所有的枚举值
            if(item.desc.equals(nameOrDesc) || item.name().equals(nameOrDesc)){
                return item; // 存在的枚举值返回
            }
        }
    }
}


//使用枚举来判断  假设type是字符串 
if(clothesType.JACKET == clothesType.parse(type))
    sout("衣服");
else if(clothesType.BOTTOMS == clothesType.parse(type))
    sout("鞋子")

枚举是可以直接使用==来判断的哦。
同样括号中的值也可以有多个”,”分割，比如根据前缀或后缀匹配做枚举转换。
